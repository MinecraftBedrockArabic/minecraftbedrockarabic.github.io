This add-on fixes the inverted syntax of the Arabic language within the game. هذا المود (إضافة او ادون) يقوم بإصلاح شكل اللغة العربية المعكوس داخل اللعبة

The add-on Features ميزات المود:

  1. Fixes the reversed syntax of Arabic text in chat
  2. Fixes the reversed syntax of Arabic text in sign
  3. Fixes the reversed syntax of Arabic text in hanging sign
  4. Fixes the reversed syntax of Arabic text in renamed items
  5. Adds a pixelated Arabic font
  1. إصلاح البنية المعكوسة للنص العربي في الرسائل
  2. إصلاح البنية المعكوسة للنص العربي في اللافتات
  3. إصلاح البنية المعكوسة للنص العربي في اللافتات المعلقة
  4. إصلاح البنية المعكوسة للنص العربي في اللغراض (الأيتمات).
  5. اضافة خط عربي مبكسل

Make sure to add new lines, so the lines order don't get reversed

تأكد من الرجوع للسطر حتى لا يتم عكس ترتيب الأسطر!

There is a bug when using glow ink, so make sure to use dye or § to color the text

يوجد خطأ عند استخدام الحبر المتوهج ، لذا تأكد من استخدام الصبغة أو § لتلوين النص

How to color text in sign:

 كيفية تلوين النص في اللافتة

How to color text in chat:

كيفية تلوين النص في الرسائل

If you want to use Arabic in commands, copy text from a fixed sign:

إذا كنت تريد استخدام اللغة العربية في الأوامر ، فقم بنسخ النص من لافتة ثم إصلاحها

You must activate Beta APIs for the add-on to work properly:

يجب عليك تفعيل Beta APIs حتى يعمل المود بشكل صحيح

After downloading the Add-on and the Resource Pack, make sure they are both activated and activate Beta APIs

بعد تنزيل المود والريسورس باك ، تأكد من تفعيلهما وتفعيل Beta APIs وبعدها يكون العالم والمود جاهز
