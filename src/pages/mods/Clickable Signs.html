This addon allows you to make signs clickable. You can also add commands that run when clicking the sign (you can add them only in creative).

Adding Commands:

⚠ Only players in creative mode can add commands!

To add commands to a sign, follow the steps bellow:

  1. Place the sign.
  2. Type "§r§r" and than the any text you want. For example: "§r§r Click me":

  1. Lock the sign with honeycomb.
  2. Interact with the sign in creative and add the commands:

Redstone Signal:

⚠ Only signs on the wall can output redstone signal!

To get a redstone signal when clicking the sign, follow the steps bellow:

  1. Place the sign on a block.
  2. If you want to only get redstone signal, add "§r" before the text and lock the sign, if you want to add commands and get redstone signal, add "§r§r" before the text:
